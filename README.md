# Laboratorio2_Grua
Este proyecto implementa un sistema de movimeinto de servos por medio de potenciometros,
cuenta con dos modos activados por botones. Dependiendo del modo, se activara un
LED, para indicar modo manual se encendera un led verde y para modo auto y de reset se enciende
un led rojo con un buzzer como alarma. Todo esta conectado a un ESP32 programado
en MicroPython.
